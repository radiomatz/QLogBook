# QLogBook
Ham Logbook with sqlite3 Database, adif im- and export and Hamlib(Net) connection

<img src="qlogbook.png">

## Qt6 based
written for linux, should run also on Windows and MacOS<br/>

## requirements:
- sqlite 3.49.100 (best compile from source)
- Qt6 (manual install)

        mkdir build
        cd build
        qmake ../qlogbook.pro
        make
